Что делать?
- перейти по ссылке, принять задание
- склонировать репозиторий к себе на компьютер через Intellij IDEA (`New->Project from Version Control->вставить URL вашего проекта->Clone`)
- дождаться индексации IDEA
- дописать необходимый код
- запустить тесты локально
- сделать коммит и запушить изменения через Intellij IDEA (`Git->Commit->выбрать осмысленное сообщение->Commit And Push`)
- зайти в репозиторий и убедиться, что тесты запустились и успешно прошли (появилась зеленая галочка)
